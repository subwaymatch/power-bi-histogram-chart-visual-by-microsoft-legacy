In 2022, Microsoft's Histogram Chart custom visual disappeared from AppSource without any official comments from Microsoft. The [official AppSource page](https://store.office.com/en-us/app.aspx?assetid=WA104380776&sourcecorrid=e26ecce9-0b9e-47f4-9c7a-a023465dafdc&searchapppos=0&ui=en-US&rs=en-US&ad=US&appredirect=false) returns a 404. This custom visual file was extracted from one of my old Power BI files.

## üß™ Loading the custom visual in Power BI

üëá Download the `.pbiviz` file below.

- [microsoft-histogram-chart-2.1.1.pbiviz](https://github.com/subwaymatch/power-bi-histogram-chart-visual-by-microsoft-legacy/releases/download/v2.2.1/microsoft-histogram-chart-2.1.1.pbiviz)
<br /><br />

üëá Click on the ellipses under the "Visualizations" section and select "Import a visual from file".

<img width="280" src="https://github.com/subwaymatch/power-bi-histogram-chart-visual-by-microsoft-legacy/assets/1064036/b4c0fc90-31ab-4c94-ac98-39f1f7fe8095">
<br /><br />

üëá Power BI may prompt a warning message. Click "Import".

<img width="640" src="https://github.com/subwaymatch/power-bi-histogram-chart-visual-by-microsoft-legacy/assets/1064036/830c6de9-c9a7-4953-b74b-37f250700a38">
<br /><br />

üëá Locate and select the `microsoft-histogram-chart-2.1.1.pbiviz` file.

<img width="480" src="https://github.com/subwaymatch/power-bi-histogram-chart-visual-by-microsoft-legacy/assets/1064036/d90f68e2-6df9-41c8-9feb-37fe230a89e1">
<br /><br />

üëá Power BI should prompt a success message.

<img width="400" src="https://github.com/subwaymatch/power-bi-histogram-chart-visual-by-microsoft-legacy/assets/1064036/c846288a-84a8-4bc0-97a9-c44c533fdf62">
<br /><br />

üëá If you see the histogram chart icon under the "Visualizations" section, the visual is ready to be used.

<img width="280" src="https://github.com/subwaymatch/power-bi-histogram-chart-visual-by-microsoft-legacy/assets/1064036/2b6d97a7-f680-4e95-abe4-0bc165576484">
